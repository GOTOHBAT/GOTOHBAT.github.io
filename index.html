idea-push-code-to-github

如何将代码托管到GitHub上

步骤

-git init
-git status
-git commit -m “(name)”
-改名字 vim .git/config
[user]
name=github用户名
email=github邮箱
-git remote add origin https://github.com/GOTOHBAT/community.git 对应的仓库
-git push -u origin master

新增文件的托管更新

-git status
-git add .
-git commit -m “” 添加备注
-链路未走完 重新commit 1.git add . 2.git commit –amend –no-edit 变化的文件追加上去
-git push
